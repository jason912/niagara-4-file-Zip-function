# niagara-4-file-Zip-function
Add file zip function to Niagara 4 station
Automated Backup and Historical Data Archiving
Historical data and reports in file format generated by a Niagara Station are often fragmented text or CSV files.
By using logic components to periodically compress old logs or reports into ZIP files, you can achieve a compression ratio of 50% to 90%.
Automatic Packaging: our jar module can automatically package the previous day's operational reports at midnight. This allows administrators to download a single archive instead of handling numerous individual files.
